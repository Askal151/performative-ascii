# Performative ASCII

Real-time generative ASCII art engine for the terminal — 15 visual layers with transitions, auto-cycle, and interactive controls.

## Layers

| Key | Layer |
|-----|-------|
| 1 | Portal Looping |
| 2 | Ball Matrix |
| 3 | Topeng Sakral |
| 4 | Matrix Data Typist |
| 5 | Tubuh Yang Mengingat |
| 6 | 3D Mountain 360 |
| 7 | Barcode Glitch |
| 8 | Wave Interference |
| 9 | Hyper Spiral |
| 10 | Void Abyss |
| 11 | Glitch Storm |
| 12 | Chaos Entropy |
| 13 | Particle Wave Field |
| `Z` | Iso Spiked Ball |
| `T` | QR Code Matrix Rain |

## Controls

| Key | Action |
|-----|--------|
| `1-9, 0, -, =, +, Z, T` | Select layer |
| `Space` / `A` | Toggle layer on/off |
| `Enter` | Toggle auto-cycle mode |
| `↑ ↓` | Select parameter |
| `← →` | Adjust parameter value |
| `C` | Toggle color mode |
| `[` `]` | Global brightness |
| `R` | Reset particles |
| `F` `S` | Transition speed |
| `H` `G` | Hold duration |
| `N` | Skip to next (auto mode) |
| `X` | Solo mode |
| `Esc` / `Q` | Quit |

## Build & Run

```bash
go build -o performative-video .
./performative-video
```
